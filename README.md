Google 模糊搜索文件采集与语种归档系统（Python）

技术栈：Python、asyncio/aiohttp/aiofiles、Redis、FastText、openpyxl/xlrd
核心职责与技术/业务落地：
•搭建Google自动化采集链路，基于关键词执行filetype:xlsx检索，解析搜索结果并提取下载链接、标题等结构化字段。
•设计异步下载模块，通过协程并发+信号量控制提升抓取吞吐，支持失败重试、临时文件落盘与异常回滚。
•构建Redis去重与断点续跑机制，按URL/MD5双重去重，支持多机共享进度，避免重复采集。
•接入域名分类与语种识别能力，通过域名规则标注站点类型，基于FastText实现语言识别与置信度过滤。
•输出标准化JSON行格式结果至Redis队列，配套调度脚本实现批处理、自动重试与异常恢复。
•完善运维能力，实现全流程消息通知，加入验证码检测与任务中断保护，提升长任务稳定性。
•开发离线后处理脚本，抽取Excel文本、进行语种分拣，实现自动归档至对应语言目录。
